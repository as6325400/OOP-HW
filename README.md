### 測試方法

為了避免有同學還不清楚，在 **cmd** 或者 **Linux** 裡，可以使用 

`./(執行檔名) < 測資 > 輸出檔 `

來進行對程式的測試，與快速取得測試結果

#### 範例
假設執行檔名稱為baseline.exe

`./baseline.exe < 1.in > 1.out`

意思就是讓 **`1.in`** 的資料流入 **`baseline.exe`** 這個程式裡 ， 輸出結果流到 **`1.out`**
